## 📌 About the Software
**FurMark 2 Pro** is an advanced GPU stress-testing and stability analysis tool designed for enthusiasts, technicians, and developers. It simulates extreme graphics workloads to test cooling performance, thermal throttling, and driver stability in real time.

## ❓ What is FurMark 2 Pro?
FurMark 2 Pro runs intensive OpenGL rendering sequences to fully load your GPU, revealing weaknesses in power delivery, cooling, or overclocking configurations. It’s a trusted diagnostic tool used by professionals to evaluate graphics performance and reliability under sustained thermal pressure.

## ✅ Why Choose FurMark 2 Pro?
- Industry-standard GPU stress-testing engine  
- Accurate temperature and power monitoring  
- Real-time graphs for load, frequency, and fan speed  
- Detects instability caused by overclocking or insufficient cooling  
- Custom benchmark presets for specific GPUs and resolutions  
- Detailed result reports for documentation and comparison  

## 🔧 Key Features
- Heavy OpenGL workload simulation for stability validation  
- Real-time GPU frequency, fan, and thermal logging  
- Integrated benchmark mode with scoring system  
- Multi-GPU support (SLI / CrossFire detection)  
- Command-line automation for lab environments  
- GPU-Z style hardware detection and monitoring  
- Optional sensor export for logging and analysis tools  
- Custom test presets for power/thermal endurance checks

## 🖥️ System Requirements
- Windows 10/11 (64-bit)  
- Dedicated GPU (NVIDIA / AMD) with OpenGL 4.6+ support  
- Latest GPU drivers installed  
- 200 MB free disk space  
- Administrator rights for performance monitoring  

## ▶️ Usage
1. Install **FurMark 2 Pro** and launch the application.  
2. Select your test preset (Resolution, MSAA, Duration).  
3. Monitor GPU temperature and power usage in real time.  
4. Run the benchmark or stress test.  
5. Save and export your report after the test completes.
